In the App:
-----------
The data directory is currently inside the project folder
It's currently having the jsons. (ex: 1.json, 2.json)


The Run with the flow.
----------------------
run app
	nodemon serveui.js

Load App in the browser
	http://localhost:3000/getdata/
	
Type 1 (which is the File Name in the system)
Click 'Load JSON'
	It should load the json in the JSON work area.
You can perform various operations on the JSON and then save it.
	(Try aditin and adding)
Click on 'Save JSON'

This will overwrite the JSON on the server in the background.
